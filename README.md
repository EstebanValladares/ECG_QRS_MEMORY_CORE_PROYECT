# ECG_QRS_Memory_Core
Este sistema se encarga de la gestión eficiente y jerárquica de la memoria de almacenamiento de los datos históricos de ECG, optimizado para consultas de análisis clínico y machine learning

### Instalacion de .venv entorno virtual
```sh
python3 -m venv .venv

```

### activacion de entorno virutal
```sh
source .venv/Scripts/activate

```

### instalacion de pymongo
```sh
pip install pymongo

```
